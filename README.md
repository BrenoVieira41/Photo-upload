### Para rodar o código

#### Cloning
```ps
  # Clone o repositório utilizando o git
  $ git clone (https://github.com/BrenoVieira41/Photo-upload.git)
```

#### :arrow_forward: Executando
```ps
  # Para baixar os pacotes
  $ npm install
  # Para executar o código
  $ npm run start:dev
```
